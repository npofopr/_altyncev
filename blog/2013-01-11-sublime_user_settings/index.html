Мой user settings в sublime text

Будучи в интернете наткнулся на настройки user конфига в sublime text и решил кое-что добавить в свой user setting.

Вот собственно мой:

	
{
	"color_scheme": "Packages/daylerees-themes/Darkside.tmTheme",
	"default_encoding": "Cyrillic (Windows 1251)",
	"detect_slow_plugins": false,
	"fallback_encoding": "Cyrillic (Windows 1251)",
	"font_face": "Consolas",
	"font_size": 8,
	"tab_size": 4,
	"translate_tabs_to_spaces": false,
	"ignored_packages":
	[
		"Vintage"
	],
	"soda_classic_tabs": true,
	"theme": "Soda Light.sublime-theme",
	"word_wrap": true,
	"caret_style": "phase",
	"highlight_line": true,
	"line_padding_bottom": 0,
	"line_padding_top": 0,
	"fade_fold_buttons": false,
	"bold_folder_labels": true
}

Немного описания (всего описывать не буду, оно и так понятно, да и при добавления в свой user settings настройки ежесукундно начинают работать):

  • color_scheme — цветовая схема
  • translate_tabs_to_spaces — я так понимаю перевод табов в пробелы
  • word_wrap — перенос слов
  • caret_style — стиль курсора
  • highlight_line — выделение строки, на которой находится курсор
  • line_padding_bottom — отступ снизу у строки (ставив просто цифру, например 1)
  • line_padding_top — естественно отступ сверху
  • fade_fold_buttons — треугольник слева, по которому можно свернуть код, теперь всегда будет на виду
  • bold_folder_labels — делает шрифт написания папок жирным, в панельке слева

Пруф линк с картиночками)
